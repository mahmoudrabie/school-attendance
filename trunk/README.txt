

Installation
============

Download the latest version of the Python GData library from
http://code.google.com/p/gdata-python-client/downloads/list.

$ tar xvf gdata-2.0.17.tar.gz
$ cd <app-home-directory>
$ ln -s <download-dir>/gdata-2.0.17/src/gdata .
$ ln -s <download-dir>/gdata-2.0.17/src/atom .

Download the latest version of the pytz library.

$ tar pytz-2012d.tar.gz
$ cd <app-home-directory>
$ ln -s <download-dir>/pytz-2012d/pytz .
